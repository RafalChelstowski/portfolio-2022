# portfolio-2022

Personal portfolio app built with React, Vite, Three.js, React Three Fiber, and Zustand.

## Requirements

- Node `24`
- npm `11` or newer

You can match the project runtime with:

```bash
nvm use
```

## Install

```bash
npm install
```

## Local development

Start the Vite dev server:

```bash
npm run dev
```

## Verification

Run the full local checks with:

```bash
npm run test
npm run typecheck
npm run lint
npm run build
```

## Available scripts

- `npm run dev` starts the local Vite dev server
- `npm run build` creates a production build in `dist/`
- `npm run lint` runs the ESLint check for the repo
- `npm run typecheck` runs TypeScript without emitting files
- `npm run test` runs the current Node test command
