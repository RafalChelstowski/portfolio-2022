- The current physics implementation lives in `src/App.tsx`, `src/features/Items.tsx`, and `src/features/Bounds.tsx` via `@react-three/cannon`.
- Current behavior to preserve: cubes spawn from the sky near `[0, 12 + index, 0]`, scale through `getSize`, highlight white on hover or presenting selection, reveal the UI after a post-startup floor collision, move toward `sortingVelocity` in global sort mode, and use center-seeking versus push-away steering for set filters.
